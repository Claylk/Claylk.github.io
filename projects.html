Trajectory Planning for Robotic Arm

Github Link

For the Robotics for Space Exploration design team (at the University of Toronto), I created a ROS node that controls a mechanical arm mounted atop our rover.

The primary goal of the node is to reduce the arm's jerk when moving between two points, ensuring safety and durability. It does this by generating a polynomial curve for each motor that drives the arm, which has zero velocity at the end points. A PID controller helps to compensate for error between calculated angles and sensor data from CAN.

The node was written for ROS 1 using Python.

Roady

For Software Design and Communication at the University of Toronto

Two groupmates and I created a GIS system (similar to Google Maps) and business pitch as part of a software design course.

The application uses the Open Street Map database and a graphics library provided by the University of Toronto (ezgl). My most challenging responsibilities for this project were implementing the A Star pathfinding algorithm, 2-opt algorithm for the travelling salesman problem, and creating the animations for the map (such as the one shown in the video).

I can't really link a GitHub here in case it gets me kicked out of school. All code was written in C++.

3 Track Digital Recorder

Github Link Demo Video

For Computer Organization at the University of Toronto

I worked with Pandu Kartosedono to create a digital recording and editing program for an Altera DE1-SoC FPGA. The program allows for 3 tracks to be recorded, layered, looped, and edited with music production in mind. The editor has highlighting / copy / paste / mute / delete functions.

All code was written in C. Graphics were doubled-buffered and made from scratch by writing to the board's video memory manually.
